A simple 2-player console-based Tic-Tac-Toe game in Python.  
Players take turns to enter moves, and the game detects win/draw automatically.  
No external libraries required, runs with Python 3.  
To play: `python tic_tac_toe.py` after cloning the repo.  
Optional: Run online via [Replit](https://replit.com/) or [Binder](https://mybinder.org/).
